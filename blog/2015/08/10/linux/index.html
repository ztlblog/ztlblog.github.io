Redis之通用key

Aug 10th, 2015 2:24 pm

redis通用key

keys

查找所用符合给定模式 pattern的key
keys * 匹配数据库中所有key
keys h?llo 匹配hello,hallo和kxllo等
keys h*llo 匹配hllo和heeeello等
keys h[ae]llo 匹配hello和hallo
特殊符合用 \ 隔开

del

如果删除的key不存在，则直接忽略
set key1 'hello'
del key1

exists

返回key是否存在
1 key存在
0 key不存在

set key1 'hello'
exists key1
(integer) 1

type

返回key所储存的值的类型

none （key不存在）
string （字符串）
list （列表）
set （集合）
zset （有序集）
hash （哈希表）

randomkey

从当前数据库返回一个随机的key

rename

将key重命名为newkey，如果key与newkey相同，将返回一个错误。
如果newkey已经存在，则值将被覆盖
